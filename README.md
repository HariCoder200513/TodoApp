# 📝 React Todo App

A simple and responsive Todo List application built using **React** and deployed using **Docker**.

## 🚀 Features

- Add, delete, and complete tasks
- Clean and responsive UI
- Dockerized for easy deployment

## 📦 Built With

- React
- HTML/CSS
- Docker
- Nginx
